# **App Name**: Project Pulse

## Core Features:

- Data Aggregation: Fetch and combine data from all Google Sheets subsheets.
- Performance Metrics Calculation: Automatically calculate performance metrics based on aggregated data (e.g., project completion time, estimation accuracy, client satisfaction).
- Trend Identification: Use AI to identify trends and patterns in project data, highlighting areas of improvement or potential issues, using internal reasoning to flag potential trends for the user. This tool does not guarantee results.
- Dashboard Visualization: Display key performance metrics and trends on an interactive dashboard, customizable by user roles and access permissions.
- Automated Reporting: Generate and send automated reports (daily, weekly, monthly) summarizing project performance to relevant stakeholders.
- Feedback Integration: Incorporate client feedback directly into the dashboard to provide a holistic view of project success.

## Style Guidelines:

- Primary color: Deep Indigo (#4B0082) to evoke a sense of professionalism and focus.
- Background color: Light Gray (#F0F0F0) for a clean and modern look.
- Accent color: Teal (#008080) to highlight key data points and calls to action.
- Body text: 'Inter' sans-serif font, offering a modern and neutral look.
- Headline text: 'Space Grotesk' sans-serif font, for a computerized, techy, scientific feel
- Use clean and professional icons to represent different metrics and data categories.
- Dashboard layout should be intuitive and easily customizable with drag-and-drop widgets.
- Subtle transitions and animations to provide a dynamic user experience.
